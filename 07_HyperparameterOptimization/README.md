## Notebooks Overview

01 - HyperparameterOptimization
Topic: A comprehensive tour of hyperparameter optimisation techniques, from classical to modern.
Dataset: Wine Quality (binary/multiclass classification).
Key Techniques: Grid Search, Random Search, Bayesian Optimisation (scikit-optimize and Optuna), Genetic Algorithm (DEAP), Particle Swarm Optimisation (pyswarms).
Visualisation: Convergence plots, bar charts comparing methods, 3D scatter of search space.
Applications: Benchmarking tuning methods, understanding trade-offs between speed and performance.

02 - BayesianOptimization
Topic: A deep dive into Bayesian optimisation for expensive black-box functions.
Dataset: Synthetic function (1D) and diabetes dataset (hyperparameter tuning of XGBoost).
Key Techniques: Gaussian process surrogate, Expected Improvement acquisition function, custom implementation with NumPy/SciPy, scikit-optimize, GPyOpt.
Visualisation: GP posterior plots, acquisition function, convergence curves.
Applications: Efficient optimisation of expensive models, hyperparameter tuning in deep learning, experimental design.

03 - ModelEnsemble
Topic: Ensemble learning for classification, combining multiple base models.
Dataset: Wine Quality (multiclass classification).
Key Techniques: Bagging (Random Forest), Boosting (AdaBoost, XGBoost), Voting (hard/soft), Stacking (with logistic regression and MLP meta-models).
Visualisation: Confusion matrices, bar charts comparing weak learners, F1-score comparisons.
Applications: Improving predictive performance, robustness, and generalisation through model combination.

## Key Learnings and Insights

### Hyperparameter Optimisation:

- Grid Search is reliable but does not scale; Random Search is more efficient for large spaces.
- Bayesian Optimisation (with GPs) is sample-efficient and outperforms random search in fewer evaluations.
- Optuna offers modern features like pruning and parallelisation, making it a go-to choice for many projects.
- Metaheuristics (GA, PSO) are powerful but require careful tuning themselves; they are useful for non-continuous or highly non-convex spaces.

### Bayesian Optimisation:

- The choice of kernel (e.g., Matern vs. RBF) and acquisition function (EI, UCB, PI) significantly affects performance.
- Implementing from scratch with NumPy/SciPy helps build intuition, while libraries like scikit-optimize and GPyOpt are more practical.

### Ensemble Methods:

- AdaBoost with moderately deep trees (depth=8) performed as well as more complex ensembles on the Wine Quality dataset.
- Voting and Stacking provide modest gains over the best single model, highlighting the importance of model diversity.
- Hyperparameter tuning (via Optuna) improved Random Forest and XGBoost, but gains were moderate – suggesting that the default parameters were already reasonable.

### Business Implications:

- Optimised models can be deployed to predict wine quality from chemical measurements, enabling quality control, production optimisation, and cost reduction.
