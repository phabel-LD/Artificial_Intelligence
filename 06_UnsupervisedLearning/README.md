Notebooks Overview

01 – Principal Component Analysis (PCA)
Topic: Linear dimensionality reduction via orthogonal projection.
Dataset: Various datasets (e.g., customer features, student performance).
Key Techniques: Standardisation, eigenvalue decomposition, explained variance ratio, scree plot, biplot.
Visualisation: 2D projection with colour‑coded classes, interactive scatter plots (Plotly), cumulative explained variance.
Applications: Noise reduction, feature extraction, visualisation of high‑dimensional data.

02 – t‑Distributed Stochastic Neighbor Embedding (t‑SNE)
Topic: Non‑linear dimensionality reduction for visualisation, preserving local structure.
Dataset: High‑dimensional datasets (e.g., customer demographics, gene expression).
Key Techniques: Perplexity tuning, early exaggeration, t‑SNE embedding in 2D/3D.
Visualisation: Interactive 2D/3D scatter plots with colour‑coded clusters.
Applications: Exploratory data analysis, visualisation of complex structures, identifying subgroups.

03 – Uniform Manifold Approximation and Projection (UMAP)
Topic: Fast, non‑linear manifold learning that preserves both local and global structure.
Dataset: High‑dimensional datasets (e.g., customer features, image embeddings).
Key Techniques: Neighbourhood graph construction, low‑dimensional embedding, tuning of n_neighbors and min_dist.
Visualisation: Interactive 2D/3D scatter plots, comparison with t‑SNE.
Applications: Scalable visualisation, pre‑processing for downstream tasks, maintaining interpretability.

04 – Exploratory Factor Analysis (EFA)
Topic: Latent variable modelling to uncover underlying factors.
Dataset: Customer behavioural data (e.g., Netflix user features).
Key Techniques: KMO measure of sampling adequacy, Bartlett's test of sphericity, factor extraction (ML or PCA), Varimax rotation, interpretation of loadings.
Visualisation: Scree plot, factor loading heatmaps, interactive factor loading plots.
Applications: Construct validation, questionnaire analysis, reducing variables to latent dimensions.

05 – K‑Means Clustering
Topic: Partitional clustering minimising within‑cluster variance.
Dataset: Numerical customer attributes (e.g., revenue, age, active days).
Key Techniques: Elbow method, silhouette analysis, optimal k selection, scaling of features.
Visualisation: Interactive 3D scatter plots with cluster assignments, silhouette plots, cluster centroids visualisation.
Applications: Customer segmentation, market basket analysis, anomaly detection.

06 – Hierarchical & Spectral Clustering
Topic: Agglomerative hierarchical clustering and graph‑based spectral clustering.
Dataset: Customer data with non‑linear structures, synthetic datasets.
Key Techniques: Linkage criteria (Ward, average, complete), dendrograms, spectral embedding using affinity matrices, eigenvalue analysis.
Visualisation: Dendrograms, 2D/3D projections of spectral embeddings, cluster assignments.
Applications: Hierarchical taxonomies, community detection, clustering non‑linearly separable data.

Key Learnings and Insights

Dimensionality Reduction:

PCA is fast and interpretable but limited to linear structures.

t‑SNE excels at visualising local neighbourhoods but is computationally expensive and stochastic.

UMAP offers a good balance of speed and structure preservation, often outperforming t‑SNE.

Factor Analysis:

Requires adequate sample size and factorability (KMO > 0.6). In the Netflix dataset, KMO was 0.50, indicating marginal suitability.

Rotation (Varimax) improves interpretability by simplifying factor loadings.

Clustering:

K‑Means is simple and effective for spherical, well‑separated clusters but fails on non‑linear data.

Hierarchical provides a dendrogram for exploring cluster hierarchies but is computationally heavy for large datasets.

Spectral Clustering captures non‑linear structures via graph Laplacians but requires careful parameter tuning (affinity, gamma, n_neighbors).

Model selection should consider both internal validation (silhouette score) and external interpretability – the best method is not always the one with the highest silhouette.

Business Applications:

The Netflix segmentation project demonstrated how K‑Modes (categorical) produced two clear, actionable clusters: a premium/engaged segment and a basic/price‑sensitive segment, directly informing targeted marketing strategies.
