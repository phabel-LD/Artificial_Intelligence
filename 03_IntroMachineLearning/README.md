## Overview

This repository contains a single, self‑contained notebook that walks through the full machine learning workflow:

- **Data Loading** – Automatically downloads the SMS Spam Collection dataset from Kaggle using `kagglehub`.
- **Exploratory Data Analysis** – Class distribution, sample inspection, basic statistics.
- **Supervised Classification** – Bernoulli Naïve Bayes, Multinomial Naïve Bayes, and Support Vector Machine (SVM) with hyperparameter tuning (GridSearchCV, RandomizedSearchCV).
- **Text Preprocessing** – Lemmatization using Stanza, comparing raw vs. preprocessed text.
- **Unsupervised Learning** – K‑means and Hierarchical clustering with T‑SNE visualisation and silhouette analysis.
- **Evaluation** – Accuracy, precision, recall, F1‑score, confusion matrices, classification reports.
- **Visualisation** – Clustering scatter plots, silhouette curves, and dendrograms.

**All code is modular, well‑documented, and adheres to PEP8 and reproducibility standards (fixed random seeds, path handling).**

## Dataset

**SMS Spam Collection** (UCI) – publicly available on Kaggle.

- **Source:** [Kaggle – SMS Spam Collection Dataset](https://www.kaggle.com/datasets/uciml/sms-spam-collection-dataset)
- **Size:** 5,574 SMS messages
- **Classes:** `spam` (unsolicited) and `ham` (legitimate)
- **Format:** Two columns: `v1` (label) and `v2` (message text)

The dataset is downloaded automatically on first run using `kagglehub` – no manual download required.
