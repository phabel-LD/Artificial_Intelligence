## Notebooks Overview

### 1. Bayesian Networks – `01_BayesianNetworks.ipynb`

**Core Concepts**: Directed Acyclic Graphs (DAGs), Conditional Probability Tables (CPTs), Variable Elimination, Exact Inference.

**Content**:
- Implementation of two classic Bayesian Networks:
  - **Alarm Network** (Burglary & Earthquake → Alarm → John/Mary calls)
  - **Sprinkler Network** (Cloudy → Sprinkler/Rain → Wet Grass)
- Library‑based inference using `pgmpy` (Variable Elimination) with detailed CPD definitions.
- **From‑scratch inference engine** for the Alarm network, computing joint and conditional probabilities purely with NumPy – demonstrating deep understanding of marginalisation and factorisation.
- DAG visualisation with `networkx` and `matplotlib`.

**Key Techniques**:
- Probabilistic graphical models, conditional independence, Bayes' rule, marginalisation.

**Visuals**:
- Publication‑quality DAG plots.
- Structured outputs (DataFrames) for transparency.

---

### 2. Hidden Markov Models – `02_HiddenMarkovModels.ipynb`

**Core Concepts**: Latent state inference, Forward algorithm (filtering), Viterbi decoding (MAP), stationary distributions.

**Content**:
- Custom HMM class implemented from scratch using only NumPy, including:
  - `forward()` – computes filtered beliefs `P(Rₜ | U₁…Uₜ)` with normalisation for numerical stability.
  - `viterbi()` – finds the most likely hidden state sequence.
  - `predict_future()` – forecasts future state distributions using the transition matrix.
- Synthetic "Weather & Umbrella" dataset with known transition/emission matrices.
- Multi‑step forecast and convergence to stationary distribution, illustrated with a clean line plot (Blue = Rain, Orange = No Rain).

**Key Techniques**:
- Dynamic programming, Bayes filtering, Markov chains, stationary analysis.

**Visuals**:
- Professional line plot with grid, legend, and equilibrium line.
