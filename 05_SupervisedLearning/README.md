## Overview

This repository contains a curated set of **seven polished Jupyter notebooks** that systematically cover the supervised learning landscape. Each notebook follows a consistent, professional structure:

- **Clear narrative** – introduction, methodology, implementation, evaluation, and critical reflection.
- **Reproducibility** – fixed random seeds, no `!pip install` cells, consistent data loading.
- **Interactive visualisations** – using Plotly for 3D and 2D plots, confusion matrices, and model diagnostics.
- **Modular code** – helper functions, docstrings, and PEP8 compliance.
- **Publication‑quality outputs** – static plots saved for portfolio inclusion.

All notebooks are designed to run in a Conda environment (`ai_env`) with Python 3.10. Dependencies are pre‑installed – ensuring a clean, professional experience.

## Notebooks Overview

### 01 – Linear Regression
**Topic:** Multiple linear regression with ordinary least squares.  
**Dataset:** Boston Housing (OpenML).  
**Key Techniques:** Train/test split, RMSE, MAE, R², coefficient interpretation, residual analysis.  
**Visualisation:** Interactive coefficient bar plots, partial regression plots, actual vs predicted scatter.  
**Applications:** Baseline regression modelling, feature importance, model diagnostics.

---

### 02 – Logistic Regression
**Topic:** Binary classification using logistic regression.  
**Dataset:** Breast Cancer Wisconsin (sklearn).  
**Key Techniques:** Feature scaling, hyperparameter tuning (GridSearchCV), ROC curve, Precision‑Recall curve.  
**Visualisation:** Pair plots, correlation heatmaps, confusion matrix, probability distributions, ROC/PR curves.  
**Applications:** Medical diagnosis, binary decision‑making, probability calibration.

---

### 03 – Multinomial Logistic Regression with Cross‑Validation
**Topic:** Multiclass classification with built‑in cross‑validation.  
**Dataset:** Vertebral Column (3 classes: Hernia, Normal, Spondylolisthesis).  
**Key Techniques:** `LogisticRegressionCV`, feature scaling, micro/macro/weighted F1‑score.  
**Visualisation:** Pair plots, confusion matrix, coefficient bar charts, probability boxplots.  
**Applications:** Multiclass medical classification, biomechanical diagnosis.

---

### 04 – Decision Trees & Random Forests
**Topic:** Tree‑based models from a single decision tree to ensembles.  
**Datasets:** Vertebral Column (multiclass) and Heart Disease (binary).  
**Key Techniques:** Decision trees, random forests, gradient boosting, GridSearchCV, feature importance.  
**Visualisation:** Static tree plots (`plot_tree`), interactive treemaps (Plotly), `supertree` interactive viewer, feature importance bar charts.  
**Applications:** Interpretable modelling, ensemble learning, clinical decision support.

---

### 05 – Ensemble Learning
**Topic:** Bagging (Random Forest) and Boosting (Gradient Boosting) compared.  
**Datasets:** Vertebral Column (multiclass) and Heart Disease (binary).  
**Key Techniques:** Cross‑validation, hyperparameter tuning, model comparison.  
**Visualisation:** Confusion matrices, feature importance plots, learning curves.  
**Applications:** Robust prediction, variance reduction, bias‑variance trade‑off.

---

### 06 – Model Selection
**Topic:** Systematic model selection with cross‑validation and hyperparameter tuning.  
**Dataset:** Pima Indians Diabetes (binary classification).  
**Key Techniques:** K‑fold cross‑validation, GridSearchCV, learning curves, model comparison (Logistic Regression, Decision Tree, Random Forest, SVM).  
**Visualisation:** Cross‑validation bar charts, confusion matrices, learning curves, feature importance.  
**Applications:** Rigorous model evaluation, hyperparameter optimisation, final model selection.

---

### 07 – Model Explainability with SHAP
**Topic:** SHAP (SHapley Additive exPlanations) for model interpretability.  
**Datasets:** Breast Cancer (logistic regression) and Loan Approval (logistic regression & decision tree).  
**Key Techniques:** SHAP summary plots, bar plots, waterfall plots, force plots, dependence plots.  
**Visualisation:** Interactive and static SHAP visualisations for global and local interpretability.  
**Applications:** Model transparency, trust building, feature impact analysis, clinical and financial decision support.
